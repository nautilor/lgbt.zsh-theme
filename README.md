## Installation

- Make sure to have [oh-my-zsh](https://ohmyz.sh/) installed

- Copy the theme inside `$HOME/.oh-my-zsh/themes/`

- Inside your `.zshrc` change `ZSH_THEME` to `lgbt`

  ```shell
  ZSH_THEME="lgbt"
  ```

- Source the `.zshrc` to reload the configuration
